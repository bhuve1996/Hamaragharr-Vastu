# Quick Setup: Create Pages and Assign Templates

## Step-by-Step Process

### ✅ Step 1: Create a New Page
1. Go to **Online Store > Pages** in Shopify Admin
2. Click **"Add page"** button (top right)
3. Fill in basic info:
   - **Title**: "About" (or "About Us")
   - **Content**: Leave blank (the template provides all content)
   - **Visibility**: Set to "Visible" (if you want it public)

### ✅ Step 2: Assign the Template
1. Scroll down on the page editor
2. Find the **"Template"** dropdown (usually near the bottom)
3. Select **`page.about`** from the dropdown
4. The template will be assigned to this page

### ✅ Step 3: Set the Page Handle (Important!)
1. Scroll to **"Search engine listing preview"** section
2. Click **"Edit website SEO"**
3. In the **"URL and handle"** field, make sure it says: `about`
   - This ensures the template applies correctly
4. Click **"Save"**

### ✅ Step 4: Save the Page
1. Click **"Save"** button (top right)
2. Your page is now created and using the `page.about` template!

## Repeat for Other Pages

### For Wishlist Page:
1. **Add page** → Title: "Wishlist"
2. **Template dropdown** → Select `page.wishlist`
3. **Handle** → Set to `wishlist`
4. **Save**

### For Contact Page:
1. **Add page** → Title: "Contact Us"
2. **Template dropdown** → Select `page.contact-us-1`
3. **Handle** → Set to `contact-us`
4. **Save**

## Summary

**Yes, you need to:**
1. ✅ Create a new page first
2. ✅ Then assign the template from the dropdown
3. ✅ Set the correct handle
4. ✅ Save

The template dropdown shows all available templates, but they need to be assigned to actual pages to be used.
